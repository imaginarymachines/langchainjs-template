# Langchain JS Starter Template

Typescript + [Langchainjs](https://js.langchain.com/docs/) starter repo

## Use


Requires Node 18+.

Install:
    - `npm i`
Create a .env file and add your OPENAI API key:
    - `OPENAI_API_KEY=sk-123...`
- Build that typescript:
    - `npm run build`
- Run that code
    - `npm run dev`
